title: Core Modules
---

이 섹션에서는 Injective 체인의 core modules를 살펴보고 온체인에 정의된 각 Message에 대한 트랜잭션 생성 및 브로드캐스트 예제를 제공합니다.

### 토픽

| 토픽                            | 설명                                          |
| -------------------------------- | ---------------------------------------------------- |
| [Auction](./examples/auction/)            | 온체인 buy-back-and-burn 메커니즘에 사용    |
| [AuthZ](./examples/authz/)                | 계정 권한 부여에 사용                |
| [Bank](./examples/bank/)                  | 사용자 자산(펀드) 관리에 사용               |
| [Distribution](./examples/distribution/)  | 온체인 배포/민팅에 사용               |
| [Exchange](./examples/exchange/)          | Exchange 프리미티브에 사용                     |
| [Feegrant](./examples/feegrant/)          | 수수료 허용 권한 부여에 사용          |
| [Governance](./examples/governance/)      | 온체인 거버넌스에 사용                         |
| [IBC](./examples/ibc/)                    | 크로스 Cosmos 체인 전송에 사용                |
| [Insurance](./examples/insurance/)        | 온체인 보험 펀드에 사용                    |
| [Peggy](./examples/peggy/)                | Injective ↔ Ethereum 브릿지에 사용            |
| [Permissions](./examples/permissions/)    | 온체인 권한에 사용
| [Staking](./examples/staking/)            | 온체인 스테이킹에 사용                            |
| [Tokenfactory](./examples/token-factory/) | `tokenfactory` 토큰 생성 및 관리에 사용 |
| [Wasm](./examples/wasm/)                  | Cosmwasm Layer와의 상호작용에 사용         |
