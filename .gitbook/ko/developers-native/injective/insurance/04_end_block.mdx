# End-Block

각 EndBlock에서 만료된 redemption 요청은 자동으로 처리되며, redemption에 대한 insurance pool token이 소각되고 redemption에 해당하는 비례 quote currency 금액이 insurance 모듈에서 underwriter의 잔액으로 인출됩니다. 자세한 내용은 [state transitions](./02_state_transitions) 섹션의 pending redemption 자동 인출에서 확인할 수 있습니다.
