# 향후 개선 사항

Chainlink, Razor, DIA, API3, UMA, Pyth를 포함한 다른 oracle provider 및 IBC를 통한 Band oracle 데이터에 대한 지원을 확장할 예정입니다.

위 항목들 중 IBC를 통한 Band 및 Pyth oracle 데이터 통합과 OCR(off-chain reporting) 메커니즘을 통한 Chainlink oracle 데이터 통합 개발이 진행 중입니다.
