Injective는 금융을 위해 구축된 Layer-1 블록체인입니다. Injective는 개발자에게 탈중앙화 금융 애플리케이션을 구축하기 위한 기본 프리미티브와 강력한 Web3 애플리케이션을 구축하기 위한 고급 기능을 제공하는 개방적이고 무허가 스마트 컨트랙트 레이어를 제공합니다.

Injective는 Ethereum, Solana 및 CosmosHub, Osmosis 등과 같은 모든 IBC 지원 cosmos 체인을 포함한 여러 잘 알려진 블록체인 네트워크와 네이티브하게 상호 운용됩니다. 이러한 상호 운용성은 Injective가 여러 체인에서 자산을 브릿지할 수 있게 할 뿐만 아니라 오라클 가격 등과 같은 임의의 데이터도 전송할 수 있게 합니다.

이 섹션에서는 Injective 위에 탈중앙화 애플리케이션을 구축할 수 있도록 `@injectivelabs` 패키지와 함께 작동하도록 다양한 UI 프레임워크를 구성하는 방법을 탐색합니다. 또한 Injective 위에 구축된 예제(간단한) dApp을 보여드리겠습니다.

보안상의 이유로 NPM 패키지의 안정 버전을 사용하는 것이 좋습니다.

### Create Injective dApp CLI 도구

Injective에서 여정을 시작하는 가장 간단한 방법은 CLI 도구를 사용하는 것입니다. 이렇게 하려면 터미널에서 이 명령을 작성하고 지침을 따르세요!

```bash
$ npx @injectivelabs/create-injective-app
```
