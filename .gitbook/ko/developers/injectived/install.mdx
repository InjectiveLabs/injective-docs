## 플랫폼 호환성 가이드

`injectived` CLI를 실행할 수 있는 플랫폼을 확인하려면 이 표를 참조하세요:

| 플랫폼 | 사전 빌드된 바이너리 | Docker | 소스에서 빌드 |
|----------|-------------------|--------|------------|
| macOS (M1/ARM) | ❌ | ✅ | ✅ |
| macOS (Intel) | ❌ | ✅ | ✅ |
| Windows (x86_64) | ❌ | ✅ | ❌ |
| Windows (ARM) | ❌ | ✅ | ❌ |
| Linux (x86_64) | ✅ | ✅ | ✅ |
| Linux (ARM) | ❌ | ✅ | ✅ |


## 사전 빌드된 바이너리로 시작하기

현재 사전 빌드된 `injectived` CLI를 실행할 수 있는 유일한 지원 플랫폼은 Linux x86_64입니다. 사전 빌드된 바이너리는 [Injective GitHub Releases 페이지](https://github.com/InjectiveLabs/injective-chain-releases/releases)에서 사용할 수 있습니다.

```bash
wget https://github.com/InjectiveLabs/injective-chain-releases/releases/download/v1.14.1-1740773301/linux-amd64.zip
unzip linux-amd64.zip
```

이 zip 파일에는 다음 파일들이 포함되어 있습니다:

* **`injectived`** - Injective 데몬 및 CLI
* **`peggo`** - Injective Ethereum 브릿지 릴레이어 데몬
* **`libwasmvm.x86_64.so`** - WASM 가상 머신 지원 파일

참고: 스마트 컨트랙트를 배포하고 인스턴스화하는 데 `peggo`는 필요하지 않습니다. 이것은 검증자를 위한 것입니다.

```bash
sudo mv injectived /usr/bin
sudo mv libwasmvm.x86_64.so /usr/lib
```

버전이 아래 출력과 일치하는지 확인하세요 (새 버전이 사용 가능한 경우 출력이 약간 다를 수 있습니다):

```bash
injectived version

Version v1.14.1 (0fe59376dc)
Compiled at 20250302-2204 using Go go1.23.1 (amd64)
```

Injective 블록체인과 상호작용하기 위해 `injectived` CLI를 사용하는 방법을 알아보려면 [injectived 사용하기](/developers/injectived/use/)를 계속 진행하세요.

## Docker로 시작하기

다음 명령은 `injectived` CLI가 포함된 컨테이너를 시작합니다:

```bash
docker run -it --rm injectivelabs/injective-core:v1.14.1 injectived version

Version v1.14.1 (0fe59376d)
Compiled at 20250302-2220 using Go go1.22.11 (amd64)
```

이것은 대부분의 플랫폼과 arm64 / x86_64 아키텍처와 호환됩니다.


Injective 블록체인과 상호작용하기 위해 `injectived` CLI를 사용하는 방법을 알아보려면 [injectived 사용하기](/developers/injectived/use/)를 계속 진행하세요.


## 소스 코드로 시작하기

다음 명령은 소스 코드에서 `injectived` CLI를 빌드합니다:

```bash
git clone https://github.com/InjectiveFoundation/injective-core.git
cd injective-core && git checkout v1.14.1
make install
```

이것은 `injectived` CLI를 go path에 설치합니다.

```bash
injectived version

Version v1.14.1 (dd7622f)
Compiled at 20250302-2230 using Go go1.24.0 (amd64)
```

(커밋 해시는 다를 수 있습니다. 오픈소스 저장소는 사전 빌드된 버전과 별도로 게시됩니다.)

Injective 블록체인과 상호작용하기 위해 `injectived` CLI를 사용하는 방법을 알아보려면 [injectived 사용하기](/developers/injectived/use/)를 계속 진행하세요.
