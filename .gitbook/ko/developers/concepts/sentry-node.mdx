Sentry 노드는 Injective 체인을 실행하는 읽기 전용 전체 노드입니다. 전체 노드는 블록체인의 트랜잭션과 블록을 완전히 검증하고 모든 역사적 활동에 대한 전체 기록을 유지하는 체인의 바이너리(소프트웨어)를 실행하는 서버입니다. 전체 노드는 블록 헤더와 트랜잭션의 작은 부분 집합만 처리하는 가지치기된 노드와 다릅니다. 전체 노드를 실행하는 것은 가지치기된 노드보다 더 많은 리소스가 필요합니다. 검증자는 전체 노드 또는 가지치기된 노드를 실행하도록 결정할 수 있지만 새 블록을 검증하기 위해 충분한 블록을 유지하도록 해야 합니다.

탈중앙화 애플리케이션에서 제공되는 온체인 데이터를 얻기 위해 sentry 노드를 쿼리합니다.
