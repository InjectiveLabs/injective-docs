# 마켓 최소 가격 tick size

주문 가격에 대한 최소 마켓 가격 tick size - 마켓의 minPriceTickSize가 `0.001`이고 가격 `0.0011`로 주문을 제출하면 거부됩니다.

spot과 quote 마켓 가격 tick size를 계산하는 공식이 다르다는 점에 유의하세요.

### Spot 마켓

1.  UI 사람이 읽을 수 있는 형식에서 체인 형식으로:
    18 base decimals와 6 quote decimals를 가진 INJ/USDT 마켓을 예로 들어, 값을 체인 형식으로 변환하는 방법입니다:

```ts
import { toChainFormat } from "@injectivelabs/utils";

const value = toChainFormat(value, quoteDecimals - baseDecimals).toFixed();
```

2. 체인 형식에서 UI 사람이 읽을 수 있는 형식으로:
   18 base decimals와 6 quote decimals를 가진 INJ/USDT 마켓을 예로 들어, 값을 UI 사람이 읽을 수 있는 형식으로 변환하는 방법입니다:

```ts
import { toHumanReadable } from "@injectivelabs/utils";

const value = toHumanReadable(value, quoteDecimals - baseDecimals).toFixed();
```

### Derivative 마켓

1.  UI 사람이 읽을 수 있는 형식에서 체인 형식으로:
    6 quote decimals를 가진 INJ/USDT perp 마켓을 예로 들어, 값을 체인 형식으로 변환하는 방법입니다:

```ts
import { toChainFormat } from "@injectivelabs/utils";

const value = toChainFormat(value, -quoteDecimals).toFixed();
```

2. 체인 형식에서 UI 사람이 읽을 수 있는 형식으로:
   6 quote decimals를 가진 INJ/USDT perp 마켓을 예로 들어, 값을 UI 사람이 읽을 수 있는 형식으로 변환하는 방법입니다:

```ts
import { toHumanReadable } from "@injectivelabs/utils";

const value = toHumanReadable(value, quoteDecimals).toFixed();
```
