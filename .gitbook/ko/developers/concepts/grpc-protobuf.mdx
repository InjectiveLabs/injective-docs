---
title: gRPC & Protobuf
---

gRPC는 모든 환경에서 실행할 수 있는 현대적인 오픈소스 고성능 원격 프로시저 호출(RPC) 프레임워크입니다. 로드 밸런싱, 추적, 헬스 체크 및 인증을 위한 플러거블 지원과 함께 데이터 센터 내부 및 데이터 센터 간에 서비스를 효율적으로 연결할 수 있습니다. 또한 기기, 모바일 애플리케이션 및 브라우저를 백엔드 서비스에 연결하는 분산 컴퓨팅의 마지막 마일에도 적용할 수 있습니다.

Protobuf는 gRPC에서 가장 일반적으로 사용되는 IDL(인터페이스 정의 언어)입니다. 기본적으로 proto 파일 형태로 데이터 및 함수 계약을 저장하는 곳입니다.

```proto
message Person {
    required string name = 1;
    required int32 id = 2;
    optional string email = 3;
}
```
