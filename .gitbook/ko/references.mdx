## 개발자 리소스

### 개발자 도구 및 리소스

Injective에서 빌드하기 위한 개발자 도구 및 리소스

| **리소스**                                                                                  | **설명**                                                                                                             |
| --------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| [MCP Server](./developers/ai/mcp)                                                               | Model Context Protocol을 사용하여 AI 어시스턴트를 Injective 문서에 연결                                           |
| [Injective 101](https://injective.notion.site/Injective-101-589dedc4c9c04531aae503dbb235d443) | Injective 리소스 원스톱샵                                                                                       |
| [Injectived](./developers/injectived/)                                                            | Injective에 연결하는 커맨드라인 인터페이스 및 노드 데몬                                                           |
| [Injective Explorer](https://explorer.injective.network/)                                     | 누구나 Injective에서 주소, 거래, 토큰, 트랜잭션 및 기타 활동을 검색할 수 있는 분석 플랫폼 |
| [Injective Local](https://github.com/InjectiveLabs/injective-local)                           | Docker로 컨테이너화되고 간단한 docker-compose 파일로 오케스트레이션된 Injective testnet 및 생태계               |
| [Injective REST API](https://lcd.injective.network/swagger/)                                  | Swagger API 탐색기                                                                                                        |
| [Injective TypeScript SDK](https://docs.ts.injective.network)                                 | TypeScript를 사용하여 Injective에서 dApp 빌드                                                                                   |
| [Injective API Reference](https://api.injective.exchange)                                     | 트레이더를 위한 Injective와 상호작용하기 위한 상세 API 문서                                                       |
| [Cosmovisor](./infra/cosmovisor/)                                                 | 거버넌스 모듈을 모니터링하는 Cosmos SDK 바이너리 주변의 소규모 프로세스 관리자                                        |
| [CosmosSDK](https://docs.cosmos.network/main/build)                                          | Injective 생태계와 통합하는 개발자를 위한 귀중한 리소스 역할을 하는 Cosmos SDK 문서         |

### 생태계 도구 및 리소스

생태계 개발자 및 파트너가 개발한 도구 및 리소스

| **리소스**                                                                                       | **설명**                                                                                                                                                                                      |
| -------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Desig](https://desig.io/)                                                                         | 다자간 연산(MPC) multisig 솔루션                                                                                                                                                      |
| [DiscoverINJ](https://alpha.discoverinj.com/)                                               | UI 스마트 컨트랙트 쿼리/실행                                                                                                                                                                    |
| [The Graph](https://docs.substreams.dev/tutorials/intro-to-tutorials/injective)                                   | Injective 블록체인에서 데이터를 추출하기 위한 Substreams                                                                                                                                        |
| [NewMetric](https://app.newmetric.xyz/)                                                            | 맞춤형 고성능 RPC 노드                                                                                                                                                                   |
| [Notifi](https://injective.com/notifications/)                                                     | Notifi를 통해 최신 체인 업데이트, 생태계 개발, 시장 인사이트 등에 대한 알림 수신                                                                                     |
| [Pyth](https://docs.pyth.network/home)                                                             | 가격 피드 oracle 및 시장 데이터                                                                                                                                                                   |
| [Starship](https://docs.cosmology.zone/starship)                                                   | 개발자가 완전히 시뮬레이션된 미니 cosmos 생태계를 구동하고 엔드투엔드 테스트 케이스를 작성할 수 있는 통합 개발 환경                                                            |
| [SubQuery](https://github.com/subquery/cosmos-subql-starter/tree/main/Injective/injective-starter) | 맞춤형 API를 제공하는 오픈 소스 데이터 인덱서                                                                                                                                                       |
| [Remix IDE](https://docs.welldonestudio.io/code/deploy-and-run/injective) | 스마트 컨트랙트 배포 및 개발 API                                                                                                                                                                                  |
| [Synternet](https://docs.synternet.com/build/data-layer/developer-portal/subscribe-to-streams)     | 실시간 라이브 데이터 스트림 구독 또는 게시                                                                                                                                                      |
| [Wormhole](https://docs.wormhole.com/wormhole)                                                     | 크로스체인 메시징 프로토콜                                                                                                                                                                       |
| [DAO DAO](https://daodao.zone/dao/injective)                                                       | DAO DAO는 블록체인에서 실행되는 기본 스마트 컨트랙트와 쉽게 상호작용할 수 있는 시각적 인터페이스를 제공하여 DAO를 구축하고 운영하는 데 도움을 줍니다. DAO DAO는 multisig 솔루션도 제공합니다. |

### 유용한 저장소

Injective에서 개발 및 통합을 위한 유용한 저장소

| **저장소**                                                                                         | **설명**                                                                                                                      |
| ------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------ |
| [CosmWasm101](https://github.com/InjectiveLabs/CosmWasm101)                                      | 2023 Injective Global Hackathon의 CosmWasm 101 프레젠테이션과 매치되는 가이드                                                 |
| [cw20-adapter](https://github.com/InjectiveLabs/cw20-adapter/tree/master/contracts/cw20-adapter) | CW-20 토큰을 Injective 체인에서 발행된 네이티브 토큰(TokenFactory 모듈 사용)으로 교환할 수 있는 컨트랙트 및 그 반대 |
| [cw-injective](https://github.com/InjectiveLabs/cw-injective)                                    | CosmWasm을 Injective와 통합하는 데 사용할 수 있는 패키지                                                                       |
| [injective-rust](https://github.com/InjectiveLabs/injective-rust)                                | Injective-Core에 대한 모든 rust 바인딩 생성기가 있는 곳                                                                   |
| [injective-ts-examples](https://github.com/InjectiveLabs/injective-ts-examples)                  | injective-ts 라이브러리로 빌드를 시작하는 데 유용한 저장소                                                                     |
| [injective-ui](https://github.com/InjectiveLabs/injective-ui)                                    | Injective에서 개발을 용이하게 하는 UI 패키지 컬렉션                                                                           |
| [swap-contract](https://github.com/InjectiveLabs/swap-contract)                                  | 두 개의 다른 토큰 간의 즉시 스왑을 보여주는 오픈 소스 원자 토큰 스왑 컨트랙트                                  |
| [test-tube](https://github.com/injectiveLabs/test-tube)                                          | CosmWasm 스마트 컨트랙트에 대한 테스트 환경을 구축하기 위한 일반 라이브러리                                                       |
