# RPC 엔드포인트

- 통합 파트너에게 권장
    - 애플리케이션 요구에 맞게 사용자 정의 가능
    - 사용자 정의 가능한 속도 제한 및 과거 데이터 쿼리에 대한 차단 없음
    - 프리미엄/유료 옵션
    - 연결 방법:
        - QuickNode: [Injective RPC Node Endpoints, APIs & Tools | QuickNode](https://www.quicknode.com/chains/inj) 
        - Thirdweb: [Injective EVM](https://thirdweb.com/injective)
- 통합 파트너에게 권장되지 않음
    - 애플리케이션 요구에 맞게 사용자 정의 불가
    - 엄격한 속도 제한 및 과거 데이터 쿼리 차단
    - 무료 옵션
    - 연결 방법:
        - JSON-RPC Endpoint: `https://sentry.evm-rpc.injective.network/`
        - WS Endpoint: `wss://sentry.evm-ws.injective.network`
    
참조: [EVM 네트워크 정보 | Injective | Docs](/ko/developers-evm/network-information#injective-evm-mainnet)

# 지갑

## `injectived`

프로그래밍 방식 제어에 적합, 예: 애플리케이션/dApp 운영 계정.

설치 빠른 시작:
  
```shell
wget https://github.com/InjectiveLabs/injective-chain-releases/releases/download/v1.17.2-1765930431/linux-amd64.zip # 최신 버전의 injective로 교체
unzip linux-amd64.zip
sudo mv injectived peggo /usr/bin
sudo mv libwasmvm.x86_64.so /usr/lib
```

<Tip>
위 명령에서 `wget` 다운로드 URL은 최신 릴리스로 교체해야 합니다.
[Mainnet Validator](/infra/validator-mainnet) 섹션에서 정확한 명령(URL 포함)을 찾을 수 있습니다.
</Tip>

<Tip>
Docker를 통한 설치 또는 소스에서 컴파일을 포함한 더 많은 옵션은
아래 참조 페이지에서 확인할 수 있습니다.
</Tip>

참조: [injectived | Injective | Docs](/developers/injectived) 

## MetaMask

EVM DApp과 상호작용해야 하는 소매 사용자에게 적합합니다.

설치:

- [https://metamask.io/en-GB](https://metamask.io/en-GB)에서 특정 브라우저용 브라우저 확장 프로그램을 받으세요.
- [https://blockscout.injective.network/](https://blockscout.injective.network/)를 방문하여 페이지 하단으로 스크롤합니다
- "Add Injective" 버튼을 클릭합니다
- MetaMask 사용자 인터페이스의 프롬프트를 따릅니다

참조: [Injective 지갑 생성 방법(MetaMask)](https://injective.com/blog/how-to-create-an-injective-wallet-2/#to-connect-to-injective-with-metamask)
