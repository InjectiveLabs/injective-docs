Injective MCP 서버를 설정하고 실행합니다.
어떤 MCP 서버를 사용할지 선택하는 데 도움을 줍니다.

<Info>
  Injective에는 2개의 MCP 서버가 있습니다:
  - [Injective Documentation MCP 서버](/ko/developers-ai/documentation-mcp/)
  - [Injective MCP 서버](/ko/developers-ai/mcp/)
</Info>

## 설치

NPX를 통한 설치:

```bash
npx skills add https://github.com/InjectiveLabs/agent-skills \
  --skill injective-mcp-servers
```

## 사용법

skill을 설치한 후, harness에 프롬프트를 입력하여 이 skill을 사용하세요.
