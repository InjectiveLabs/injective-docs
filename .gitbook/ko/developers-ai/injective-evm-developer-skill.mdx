Injective에서 EVM 스마트 컨트랙트 및 dApp을 개발하세요.

## 설치

NPM을 통해 현재 디렉토리에 설치, 현재 프로젝트에서만 사용:

```bash
npx skills add InjectiveLabs/agent-skills --skill injective-evm-developer
```

NPM을 통해 전역 설치, 모든 프로젝트에서 사용:

```bash
npx skills add InjectiveLabs/agent-skills --global --skill injective-evm-developer
```

## 사용법

샘플 프롬프트는 skill 디렉토리 내의 `references/sample-prompts.md`를 참조하세요.

skill을 설치한 후, harness에 프롬프트를 입력하여 이 skill을 사용하세요.
