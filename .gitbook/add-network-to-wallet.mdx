Add Injective to your browser wallet with a single click, or use the manual
configuration details below. Injective exposes two interoperable interfaces:

- **Injective EVM** is an Ethereum-compatible JSON-RPC layer. Use it with EVM
  wallets such as MetaMask, Rabby, or any injected `window.ethereum` wallet.
- **Injective (CosmWasm / native)** is the native Cosmos SDK chain. Use it with
  Cosmos wallets such as Keplr and Leap.

<Info>
Injective EVM and Injective native map to the same network. They use different
chain IDs but share the same underlying state. See
[Network Information](/developers/network-information) and
[EVM Network Information](/developers-evm/network-information) for the full reference.
</Info>

export const AddNetworkButton = ({ label, logo, kind, config }) => {
  const handleClick = async () => {
    try {
      if (kind === "evm") {
        const flag = config.flag;
        let provider = null;
        if (flag === "isSubWallet" && window.SubWallet) {
          provider = window.SubWallet;
        } else {
          const eth = window.ethereum;
          if (eth) {
            const list = eth.providers && eth.providers.length ? eth.providers : [eth];
            provider = list.find((p) => p && p[flag]) || eth;
          }
        }
        if (!provider) {
          alert("Please install " + label + " or another EVM wallet.");
          return;
        }
        try {
          await provider.request({ method: "eth_requestAccounts" });
        } catch (e) {
          console.error("Wallet connection rejected:", e);
          return;
        }
        try {
          await provider.request({
            method: "wallet_addEthereumChain",
            params: [config.chain],
          });
        } catch (addError) {
          try {
            await provider.request({
              method: "wallet_switchEthereumChain",
              params: [{ chainId: config.chain.chainId }],
            });
          } catch (switchError) {
            console.error("Failed to add or switch to Injective:", addError, switchError);
          }
        }
      } else if (kind === "cosmos") {
        const provider = config.wallet === "keplr" ? window.keplr : window.leap;
        if (!provider) {
          alert("Please install the " + label + " wallet.");
          return;
        }
        await provider.enable(config.chainId);
      }
    } catch (e) {
      console.error("Wallet action failed:", e);
    }
  };
  return (
    <button
      type="button"
      onClick={handleClick}
      style={{
        display: "inline-flex",
        alignItems: "center",
        gap: "9px",
        padding: "9px 16px 9px 10px",
        borderRadius: "11px",
        border: "1px solid rgba(128,128,128,0.28)",
        background: "transparent",
        color: "inherit",
        fontWeight: 600,
        fontSize: "14px",
        cursor: "pointer",
        minWidth: "130px",
        marginRight: "10px",
        marginBottom: "10px",
      }}
    >
      <img src={logo} alt={label} width="26" height="26" style={{ borderRadius: "7px", flexShrink: 0 }} />
      <span>{label}</span>
    </button>
  );
};

## Injective EVM

Injective EVM works with EVM wallets like MetaMask and Rabby. Pick a network and
click your wallet. It will prompt you to add or switch to Injective.

<NetworkCard title="Injective EVM Mainnet" chainId="1776">
  <AddNetworkButton label="MetaMask" logo={METAMASK_LOGO} kind="evm" config={{ flag: "isMetaMask", chain: EVM_MAINNET }} />
  <AddNetworkButton label="SubWallet" logo={SUBWALLET_LOGO} kind="evm" config={{ flag: "isSubWallet", chain: EVM_MAINNET }} />
</NetworkCard>

<NetworkCard title="Injective EVM Testnet" chainId="1439">
  <AddNetworkButton label="MetaMask" logo={METAMASK_LOGO} kind="evm" config={{ flag: "isMetaMask", chain: EVM_TESTNET }} />
  <AddNetworkButton label="SubWallet" logo={SUBWALLET_LOGO} kind="evm" config={{ flag: "isSubWallet", chain: EVM_TESTNET }} />
</NetworkCard>

<Info>
The button uses the standard `wallet_addEthereumChain` and
`wallet_switchEthereumChain` requests. If the network is already added, your
wallet shows a switch prompt instead. If nothing happens, add it manually with
the values below.
</Info>

## Injective (CosmWasm / Native)

The native Injective chain works with Cosmos wallets like Keplr and Leap.
Injective ships built into both wallets, so it is available out of the box.
Pick a network and click your wallet to unlock the chain.

<NetworkCard title="Injective Mainnet" chainId="injective-1">
  <AddNetworkButton label="Keplr" logo={KEPLR_LOGO} kind="cosmos" config={{ wallet: "keplr", chainId: "injective-1" }} />
  <AddNetworkButton label="Leap" logo={LEAP_LOGO} kind="cosmos" config={{ wallet: "leap", chainId: "injective-1" }} />
</NetworkCard>

<NetworkCard title="Injective Testnet" chainId="injective-888">
  <AddNetworkButton label="Keplr" logo={KEPLR_LOGO} kind="cosmos" config={{ wallet: "keplr", chainId: "injective-888" }} />
  <AddNetworkButton label="Leap" logo={LEAP_LOGO} kind="cosmos" config={{ wallet: "leap", chainId: "injective-888" }} />
</NetworkCard>

<Info>
Keplr and Leap include Injective by default, so the button simply unlocks the
chain in your wallet. For a full wallet integration guide, see
[Wallets](/defi/wallet/index).
</Info>
