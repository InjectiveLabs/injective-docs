This section covers the different ways you can interact with Injective nodes:

- [Command line](command-line) - Use the `injectived` CLI to interact with a node
- [gRPC](grpc) - Use gRPC for programmatic access and debugging
- [Go](go) - Interact with nodes programmatically using Go
- [REST](rest) - Use REST endpoints for convenient HTTP-based queries
