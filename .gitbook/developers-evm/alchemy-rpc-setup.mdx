[Alchemy](https://www.alchemy.com/) provides reliable JSON-RPC endpoints for the Injective EVM mainnet and testnet. Use this guide to provision an Alchemy app and connect it to Injective.

## Prerequisites

* An email address to sign up for Alchemy
* Basic familiarity with EVM RPC endpoints

## Steps

<Steps>

<Step title="Create an Alchemy account">
Sign up for free at [alchemy.com](https://www.alchemy.com/).
</Step>

<Step title="Create an app">
From the Alchemy dashboard, create a new app and select **Injective** as the chain. Choose the network you want to target:

* **Injective EVM Mainnet** — Chain ID `1776`
* **Injective EVM Testnet** — Chain ID `1439`
</Step>

<Step title="Get your API key">
Open your app and copy the unique HTTPS and WebSocket endpoints. They follow this format:

```bash Mainnet
https://inj-mainnet.g.alchemy.com/v2/<YOUR_API_KEY>
wss://inj-mainnet.g.alchemy.com/v2/<YOUR_API_KEY>
```

```bash Testnet
https://inj-testnet.g.alchemy.com/v2/<YOUR_API_KEY>
wss://inj-testnet.g.alchemy.com/v2/<YOUR_API_KEY>
```

Keep your API key private — treat it like a password.
</Step>

<Step title="Choose your chain">
Select either Injective EVM Mainnet or Testnet in your app settings, and reference the [Injective EVM network information](/developers-evm/network-information) for chain IDs, explorers, and other network details.
</Step>

<Step title="Start building">
Use Alchemy's SDKs, JSON-RPC API, and tooling to build your dApp. A minimal example with [ethers.js](https://docs.ethers.org/):

```ts
import { JsonRpcProvider } from "ethers";

const provider = new JsonRpcProvider(
  "https://inj-mainnet.g.alchemy.com/v2/<YOUR_API_KEY>"
);

const blockNumber = await provider.getBlockNumber();
console.log("Latest block:", blockNumber);
```

Or with `curl`:

```bash
curl https://inj-mainnet.g.alchemy.com/v2/<YOUR_API_KEY> \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"eth_blockNumber","params":[],"id":1}'
```
</Step>

</Steps>
