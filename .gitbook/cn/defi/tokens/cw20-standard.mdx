---
title: CW20 标准
---

CW20 token 标准提供了一个无需许可创建和管理同质化 token 的框架，更接近于 [ERC20 标准](https://ethereum.org/en/developers/docs/standards/tokens/erc-20/)。如上所述，由于 TokenFactory 与 Cosmos SDK 的原生集成，我们鼓励使用它，但如果你出于任何原因希望使用 CW20 标准，你可以使用 [CW20 Adapter](https://github.com/CosmWasm/cw-plus/blob/main/packages/cw20/README.md) 将 CW20 token 转换为 TokenFactory token，反之亦然。有关 CW20 标准的更多信息，请参阅其正式规范[这里](https://github.com/CosmWasm/cw-plus/blob/main/packages/cw20/README.md)。
