设置并运行 Injective MCP servers。
协助选择使用哪个 MCP server。

<Info>
  请注意，Injective 有 2 个 MCP servers:
  - [Injective Documentation MCP Server](/cn/developers-ai/documentation-mcp/)
  - [Injective MCP Server](/cn/developers-ai/mcp/)
</Info>

## 安装

通过 NPX 安装:

```bash
npx skills add https://github.com/InjectiveLabs/agent-skills \
  --skill injective-mcp-servers
```

## 使用方法

安装 skill 后，在 harness 中输入你的 prompt 即可使用此 skill。
