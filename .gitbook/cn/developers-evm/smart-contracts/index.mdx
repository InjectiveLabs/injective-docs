智能合约是在区块链上运行的代码。
你可以编译 Solidity 智能合约，
并在 Injective 的 Ethereum Virtual Machine (EVM) 上进行测试、部署、验证和交互。
你也可以使用标准的 EVM 开发工具和框架来完成这些操作。
以下指南将向你展示如何使用 Hardhat 和 Foundry 来实现。
