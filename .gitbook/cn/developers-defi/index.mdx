Injective 是专注于交易所的 Layer-1 区块链的前沿，提供去中心化永续合约、期货和现货交易。它充分利用了去中心化衍生品和无边界 DeFi 的可能性。每个组件都设计为完全无需信任、抗审查、可公开验证和抗抢跑。

Injective 使交易者能够创建和交易任意现货和衍生品市场。Injective 还通过 Injective 链的 [Exchange 模块](/developers-native/injective/exchange) 编码的逻辑实现链上限价订单簿管理、链上交易执行、链上订单匹配、链上交易结算和链上交易激励分配。

## 在 Injective 上构建订单簿 DEX

作为鼓励交易所在 Injective 上构建和获取交易活动的激励机制，将订单发起到 Injective 交易所协议共享订单簿的交易所（[阅读更多](/developers-native/injective/exchange)）将获得其发起订单交易费用的 $β=40%β=40%$ 作为奖励。交易所协议实施了全球最低交易费用，maker 为 $$rm=0.1%rm​=0.1%$$，taker 为 $$rt=0.2%rt​=0.2%$$。

Injective 激励机制的目标是允许交易所相互竞争，以提供更好的用户体验和更好地服务用户，为全球用户扩大 DeFi 的访问范围。

交易所可以轻松设置客户端（如 Web 或移动端的 UI）和 API 提供商。
