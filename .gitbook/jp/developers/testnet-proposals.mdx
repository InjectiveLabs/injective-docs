---
title: Testnet Proposals
updatedAt: "2025-10-30"
---

`testnet` で proposal を提出したい場合を考えてみましょう。proposal の投票期間が短いため、proposal が直接投票ステージに進まないように、proposal の deposit を下げることをお勧めします。基本的には、`min_deposit` の値よりもわずかに少ない金額にするべきです。

proposal を提出したら、チームに連絡してください:

1. [Injective Discord server](https://discord.gg/injective) に参加し、該当するチャンネルを見つけてください。
2. [Injective Developer Telegram channel](https://t.me/+8Y_0HOFLhnRlZDU9) に参加してください。

以下は `GrantProviderPrivilegeProposal` の例です

```bash
injectived tx oracle grant-provider-privilege-proposal YOUR_PROVIDER \
  YOUR_ADDRESS_HERE \
  --title="TITLE OF THE PROPOSAL" \
  --description="Registering PROVIDER as an oracle provider" \
  --chain-id=injective-888 \
  --from=local_key \
  --node=https://testnet.sentry.tm.injective.network:443 \
  --gas-prices=160000000inj \
  --gas=20000000 \
  --deposit="40000000000000000000inj" <-- use this amount
```
