Injective は、クロスチェーン取引、デリバティブ、DeFi、および Web3 アプリケーションのために特別に設計された唯一のブロックチェーンです。
DeFi エコシステムビルダーの世界的な中心地となるべく位置づけられた Injective は、
開発者に数多くの利点を提供し、より短い時間でより強力なアプリケーションを構築できるようにします。

このセクションでは、Injective 上で構築を行う開発者向けに、包括的なガイド、ツール、およびリソースを提供します。
Solidity または CosmWasm でスマートコントラクトを作成する場合でも、分散型取引所を開発する場合でも、
ネイティブモジュールと統合する場合でも、フルスタック dApp を構築する場合でも、
このドキュメントは迅速に開始し、Injective の DeFi に特化した独自のインフラを活用する助けとなります。

Injective は、Ethereum と Cosmos 両方のエコシステムの長所を組み合わせ、デュアル スマートコントラクト環境を提供します。
Solidity 開発者向けの EVM と Rust 開発者向けの CosmWasm を提供しつつ、
コアな金融インフラをゼロから再構築する必要をなくすネイティブ DeFi モジュールを備えています。
650ms という超高速のブロック生成時間、$0.0003 のトランザクションコスト、MEV 耐性のあるインフラ、
そして Ethereum や Solana を含む 23 以上のネットワークとのシームレスな相互運用性により、
開発者はパフォーマンス、セキュリティ、ユーザーエクスペリエンスを損なうことなく、
高度なクロスチェーン金融アプリケーションを構築できます。

## 主要概念

| 用語 | 説明 |
|------|-------------|
| **injectived** | Injective ブロックチェーンに接続して対話するためのコマンドラインインターフェースおよびノードデーモンです。 |
| **Denom** | Injective 上のネイティブアセット表現フォーマットです（例: `factory/{creator}/{subdenom}` または `peggy{ethereumAddress}`）。 |
| **Trading Account** | Bank モジュールから資金を切り離して取引を実行するための分離されたサブアカウントです。1 つのアドレスは無制限の trading account を持つことができます。 |
| **Token Factory** | ユーザーやコントラクトが新しいネイティブトークンを作成し、ネイティブトークンと CW20 トークンの間でスワップできるネイティブモジュールです。 |
| **Indexer API** | チェーンからの高速クエリ、ストリーミング更新、および履歴データを提供する高性能データインデックスサービスです。 |
