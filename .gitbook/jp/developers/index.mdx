Injectiveは、クロスチェーントレーディング、デリバティブ、DeFi、Web3アプリケーションに特化して設計された唯一のブロックチェーンです。
DeFiエコシステムのグローバルハブとして位置づけられており、開発者がより強力なアプリケーションを短期間で構築できるよう、多くの利点を提供します。

このセクションでは、Injective上で開発を行うための包括的なガイド、ツール、リソースを提供します。
SolidityやCosmWasmでスマートコントラクトを作成する場合、分散型取引所を開発する場合、
ネイティブモジュールと統合する場合、フルスタックのdAppを構築する場合、
このドキュメントを活用することで、すぐに開発を開始し、InjectiveのDeFi特化型インフラストラクチャを活用できます。

Injectiveは、EthereumとCosmos両エコシステムの強みを融合し、デュアルスマートコントラクト環境を提供します：
・Solidity開発者向けのEVM
・Rust開発者向けのCosmWasmさらに、ネイティブDeFiモジュールを備えているため、
コアとなる金融インフラストラクチャをゼロから再構築する必要がありません。
超高速の650msブロックタイム、約$0.0003のトランザクションコスト、MEV耐性インフラ、そしてEthereumやSolanaを含む23以上のネットワークとのシームレスな相互運用性により、開発者はパフォーマンス・セキュリティ・UXを犠牲にすることなく、高度なクロスチェーン金融アプリケーションを構築できます。

## 主要な概念

| 用語 | 説明 |
|------|-------------|
| **injectived** | Injectiveブロックチェーンに接続・操作するためのCLIおよびノードデーモンです。 |
| **Denom** | Injective上のネイティブアセット表現形式です（例: `factory/{creator}/{subdenom}` または `peggy{ethereumAddress}`）。 |
| **Trading Account** | トレードを実行するためにBankモジュールから資金を分離した独立したサブアカウントです。1つのアドレスで無制限の数のトレーディングアカウントを持つことができます。 |
| **Token Factory** | ユーザーやコントラクトが新しいネイティブトークンを作成し、ネイティブトークンとCW20トークン間でスワップできるネイティブモジュールです。 |
| **Indexer API** | チェーンから高速クエリ、ストリーミング更新、履歴データを提供する高性能なデータインデックスサービスです。 |
