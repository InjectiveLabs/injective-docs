Injective は、金融のために構築された Layer-1 ブロックチェーンです。Injective は、堅牢な Web3 アプリケーションを構築するための高度な機能を提供するオープンかつパーミッションレスなスマートコントラクトレイヤーに加え、分散型金融アプリケーションを構築するための即時利用可能なプリミティブを開発者に提供します。

Injective は Ethereum、Solana、および CosmosHub、Osmosis などのすべての IBC 対応 cosmos chain を含むいくつかの有名なブロックチェーンネットワークとネイティブに相互運用可能です。相互運用性により、Injective は複数のチェーンからアセットをブリッジすることをユーザーに可能にするだけでなく、oracle prices などの任意のデータの転送も可能にします。

このセクションでは、`@injectivelabs` パッケージと連携するように異なる UI フレームワークを設定する方法を探り、Injective 上で分散型アプリケーションの構築を始めることができるようにします。また、Injective 上で構築されたサンプル（シンプルな）dApp もご紹介します。

セキュリティ上の理由から、NPM パッケージの安定版パッケージバージョンを使用することをお勧めします。

### Create Injective dApp CLI ツール

Injective での開発を始める最も簡単な方法は、私たちの CLI ツールを使用することです。これを行うには、単純にこのコマンドを入力して、ターミナルの指示に従ってください！

```bash
$ npx @injectivelabs/create-injective-app
```
