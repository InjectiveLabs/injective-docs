---
title: Sentry Node
updatedAt: "2025-10-30"
---

Sentry node は、Injective chain を実行する読み取り専用のフルノードです。フルノードとは、チェーンのバイナリ（そのソフトウェア）を実行しているサーバーで、ブロックチェーンのトランザクションとブロックを完全に検証し、すべての過去のアクティビティの完全な記録を保持します。フルノードは、ブロックヘッダーとトランザクションの小さなサブセットのみを処理する pruned ノードとは異なります。フルノードの実行には、pruned ノードよりも多くのリソースが必要です。Validator は、フルノードまたは pruned ノードのいずれを実行するかを決定できますが、新しいブロックを検証できるように十分なブロックを保持する必要があります。

分散型アプリケーションで配信されるオンチェーンデータを取得するために、sentry node にクエリを実行します。
