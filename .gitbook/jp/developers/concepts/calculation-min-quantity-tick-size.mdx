---
title: Market min quantity tick size の計算
updatedAt: "2025-12-23"
---

注文価格に対する市場の最小 quantity tick size - 市場の minQuantityTickSize が `0.001` で、`0.0011` の数量での注文送信は拒否されます。

derivative 市場は UI とチェーンの間で minQuantityTickSize について同じ形式を共有するため、フォーマットは不要です。

## Spot market

1.  UI ヒューマンリーダブル形式からチェーン形式へ:
    18 base decimals と 6 quote decimals を持つ INJ/USDT 市場を例として、値をチェーン形式に変換する方法は以下のとおりです:

```js
import { toChainFormat } from "@injectivelabs/utils";

const chainFormat = toChainFormat(value, baseDecimals).toFixed();
```

2. チェーン形式から UI ヒューマンリーダブル形式へ:
   18 base decimals と 6 quote decimals を持つ INJ/USDT 市場を例として、値を UI ヒューマンリーダブル形式に変換する方法は以下のとおりです:

```js
import { toHumanReadable } from "@injectivelabs/utils";

const humanReadableFormat = toHumanReadable(
  minQuantityTickSize,
  baseDecimals
).toFixed();
```
